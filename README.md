# Tycoon New York Classic

Um protótipo simples de jogo estilo Tycoon ambientado em Nova York.  
O objetivo é construir negócios, administrar recursos e expandir seu império pela cidade.

---

## 🎮 Como jogar

1. Abra o arquivo `index.html` em qualquer navegador moderno.
2. Clique em **Iniciar Jogo** para começar.
3. Use os botões para abrir novos negócios e acompanhar seu dinheiro.
4. Expanda seu império sem falir!

## 🚀 Publicação no GitHub Pages

1. Vá em **Settings > Pages** no repositório.
2. Selecione a branch `main` e a pasta `/root`.
3. Clique em **Save**.
4. O jogo ficará disponível em:  
   `https://seu-usuario.github.io/tycoon-new-york-classic`
